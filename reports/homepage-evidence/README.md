# Homepage Evidence Package

This folder contains the ready-to-share package for NICSI ERP Portal homepage evidence.

## Contents
- `WEBSITE_FEATURES_AND_VIEWS.md` - official full homepage view document.
- `capture-checklist.md` - screenshot capture checklist.
- `templates/submission-note-template.md` - submission message template.
- `ROADMAP_EXECUTIVE_ONE_PAGE.md` - one-page strategy brief.
- `ROADMAP_DETAILED_PRESENTATION.md` - slide-wise detailed roadmap.
- `public/reports/screenshots/full-homepage.png` - place final screenshot with this exact name.

## Required Action
1. Capture one full homepage long screenshot (desktop).
2. Save it as `public/reports/screenshots/full-homepage.png`.
3. Share `WEBSITE_FEATURES_AND_VIEWS.md` with screenshot.
